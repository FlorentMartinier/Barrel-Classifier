🛢️ Barrel Manager
L'application ultime pour le suivi et la science du micro-vieillissement en fûts.

Barrel Manager est une application Android conçue pour les distillateurs amateurs, les vignerons et les passionnés de spiritueux qui souhaitent suivre avec précision l'évolution de leurs breuvages en fûts. Que vous utilisiez des fûts traditionnels ou des systèmes modernes, cette application centralise toutes vos données de vieillissement.

✨ Fonctionnalités clés
- Gestion de Chai : Suivez plusieurs fûts simultanément avec une vue Grille ou Liste détaillée.
- Historique Complet : Enregistrez chaque événement (remplissage, ouillage, dégustation, prise de photo).
- Statistiques Avancées : Visualisez l'évolution de vos spiritueux via des graphiques (Taux d'alcool, Part des Anges, extraction des tanins).
- Export PDF & Cloud : Exportez vos fiches de fûts en PDF ou synchronisez vos données via QR Code.
- A venir : Support des Fûts Hybrides - Pensée pour les nouvelles tendances comme les fûts Ten 30 ou Badmotivator, permettant de simuler le vieillissement d'un fût de 200L dans de petits contenants.

🛠️ Stack Technique
Le projet est en constante évolution, avec un focus particulier sur la performance et l'expérience utilisateur :
- Langage : Kotlin
- Architecture : MVVM / Clean Architecture
- UI : XML (Migration souhaitée vers Jetpack Compose)
- Base de données : SQLite (Room)
- Images : Coil (Coroutines Image Loader)
- Graphiques : MPAndroidChart
